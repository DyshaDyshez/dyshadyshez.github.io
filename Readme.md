# Промт-мастер ✨

[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://dyshadyshez.github.io/Promt-Changer/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Промт-мастер** — это бесплатный онлайн-инструмент для улучшения промтов (запросов) к нейросетям. Автоматически переводит на английский, добавляет детали, стили и теги для идеальных запросов в Midjourney, ChatGPT, DALL-E, Stable Diffusion и другие генеративные модели.

## Возможности 🚀

- 🔮 **Автоматическое улучшение** — превращает сырую идею в профессиональный промт
- 🌐 **Автоперевод** — для изображений и видео промты генерируются на английском (лучше понимают нейросети)
- 🎨 **9 направлений** — изображения, текст, код, Excel, видео, музыка, 3D, дизайн, маркетинг
- 🏷️ **Специальные теги** — для кода (один HTML-файл, адаптивность), Excel (формулы, макросы), маркетинга (SEO, CTA)
- 📐 **Соотношение сторон** — 16:9, 1:1, 9:16, 21:9 для визуальных промтов
- 🎚️ **Уровень детализации** — от 1 до 5
- 🌙 **Тёмная тема** — с поддержкой системных настроек
- 📋 **Копирование** — одной кнопкой
- 🖱️ **3D параллакс** — красивые левитирующие фигуры

## Как использовать 📝

1. Введите сырую идею (например, "Котик Киберпанк (стиль можно выбрать кнопками))
2. Выберите направление (картинка, текст, код и т.д.)
3. Выберите стиль (он автоматически подстраивается под направление)
4. Настройте дополнительные параметры:
   - Контекст / детали
   - Уровень детализации
   - Теги (появляются для некоторых направлений)
   - Соотношение сторон (для визуальных)
5. Нажмите **"Улучшить промт"**
6. Скопируйте результат и используйте в любимой нейросети!

## Для каких нейросетей подходит 🤖

- **Изображения:** Midjourney, DALL-E 2/3, Stable Diffusion, Adobe Firefly, Kandinsky
- **Текст:** ChatGPT, Claude, Gemini, YandexGPT, GigaChat
- **Код:** GitHub Copilot, Codeium, Cursor
- **Excel:** Любые запросы для формул и макросов

## Технологии ⚙️
